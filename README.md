# JavaScript Functions Practice

A simple project to practice fundamental JavaScript functions, including regular functions, arrow functions, parameters, return values, and console output.

## 📌 Overview

This project demonstrates:

- Regular Functions
- Arrow Functions
- Function Parameters
- Return Values
- Console Output

## 🧰 Features

1. **Calculate Age in Days**  
   Convert a person's age in years to total days using a function.

2. **Calculate Final Price with Tax and Discount**  
   Compute the final price of an item after adding tax and subtracting optional discount.

3. **Calculate Average Score**  
   Compute the average of three subject scores using an arrow function.

4. **Check if a Person is an Adult**  
   Returns `true` if age is 18 or older, otherwise `false`.
